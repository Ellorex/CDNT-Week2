Intro

Quand sa portée n'est pas précisée, une propriété est par défaut publique. Deux paires d'accolades permettent d'accéder à la valeur associée à une propriété. Une paire de parenthèses dans un élément HTML permet d'injecter un événement (directive).

Pour typer au moment de la déclaration, propriété: typage = valeur; exemple : nb1: number = 17;
5 types primitifs : string, number, boolean, undefined & null. Pas de majuscule pour les types primitifs.
Pour les instanciations de class : propriété: Class = valeur;
Incohérence non bloquante, ES5 ignore les types. Les types sont facultatifs
Une classe par fichier

Voc

  • linter : vérificateur de code de l'éditeur
  • Contrôle (UI): tout élément présent par exemple dans un formulaire client
  • One-way data binding : lorsque la récupération / le transfert d'une donnée n'est possible que dans un sens (que de la classe vers la vue ou l'inverse)
